# 矿产（Minerals）汇报材料配色规范

## 一、设计理念

矿产的视觉识别体系体现**稳重、专业、权威**的企业形象，配色方案融合了：
- **矿业基因**：深棕、土黄等大地色系，呼应矿产资源行业属性
- **国资气质**：沉稳低调的色调，体现央企的权威性与可信度
- **现代专业**：克制的色彩运用，避免花哨，强调内容本身

---

## 二、主色板（Primary Palette）

### 1. 矿业金（Mining Gold）
```
HEX: #B89A4B
RGB: 184, 154, 75
Pantone: 7555 C
```
**用途**：标题、强调文字、关键数据、图标点缀
**心理暗示**：价值、财富、品质、权威

### 2. 深棕褐（Deep Brown）
```
HEX: #6B4423
RGB: 107, 68, 35
Pantone: 7516 C
```
**用途**：副标题、次级强调、边框装饰
**心理暗示**：稳重、可靠、历史感、根基

### 3. 石墨灰（Graphite Gray）
```
HEX: #4A4A4A
RGB: 74, 74, 74
Pantone: Cool Gray 11 C
```
**用途**：正文文字、说明文字、图表标签
**心理暗示**：专业、中性、可读性强

---

## 三、辅助色板（Secondary Palette）

### 4. 浅米白（Cream White）
```
HEX: #F5F0E6
RGB: 245, 240, 230
Pantone: 7499 C
```
**用途**：页面主背景、卡片底色、大面积留白
**心理暗示**：干净、开阔、温和、不刺眼

### 5. 浅灰卡（Light Gray Card）
```
HEX: #E8E4DC
RGB: 232, 228, 220
Pantone: 7499 C (10% darker)
```
**用途**：卡片背景、模块区分、表格底色
**心理暗示**：层次、结构化、柔和过渡

### 6. 边框灰（Border Gray）
```
HEX: #D4CFC4
RGB: 212, 207, 196
```
**用途**：边框线、分隔线、轻量边框
**心理暗示**：边界、秩序、不抢镜

## 四、功能色（Functional Colors）

用于状态标识、数据可视化、强调提示：

| 功能 | 色名 | HEX | 用途 |
|------|------|-----|------|
| 成功/完成 | 墨绿 | #2D5A3D | 已完成事项、正向数据 |

## 五、配色应用规范

### 1. 页面层级配色
```
页面背景: #F5F0E6 (浅米白)
├── 顶部导航/标题区: #6B4423 (深棕褐) + #B89A4B (矿业金)文字
├── 内容卡片: #FFFFFF 或 #E8E4DC (浅灰卡)
│   ├── 卡片标题: #6B4423 (深棕褐)
│   ├── 正文内容: #4A4A4A (石墨灰)
│   └── 强调数据: #B89A4B (矿业金)
└── 底部/页脚: #4A4A4A (石墨灰) 或 #6B4423 (深棕褐)
```

### 2. 文字配色规范

| 文字类型 | 颜色 | 字重 | 用途 |
|---------|------|------|------|
| 大标题 | #6B4423 | 700-900 | 页面主标题 |
| 小标题 | #B89A4B | 600-700 | 模块标题、卡片标题 |
| 正文 | #4A4A4A | 400-500 | 说明文字、内容正文 |
| 辅助文字 | #8A8A8A | 400 | 日期、注释、次要信息 |
| 链接/强调 | #B89A4B | 500-600 | 可点击文字、关键数据 |

### 3. 图表/数据可视化配色

```
主色调序列（推荐用于柱状图、折线图）:
1. #B89A4B (矿业金) - 主要数据
2. #6B4423 (深棕褐) - 次要数据
3. #4A6B8A (钢蓝) - 辅助数据
4. #8A8A8A (中性灰) - 对比/基准数据

状态色（用于饼图、状态指示）:
- 完成/正常: #2D5A3D (墨绿)
- 进行中/警告: #B87333 (赭石)
- 异常/风险: #8B3A3A (暗红)
```

### 4. 边框与分隔线

```
主要边框: 1px solid #D4CFC4 (边框灰)
强调边框: 2px solid #B89A4B (矿业金) 或 #6B4423 (深棕褐)
内部分隔线: 1px solid #E8E4DC (浅灰卡)
阴影效果: box-shadow: 0 2px 8px rgba(107, 68, 35, 0.08)
```

## 六、配色使用禁忌

### ❌ 禁止使用

1. **高饱和度亮色**: 避免使用 #FF0000、#00FF00 等高纯度色，破坏专业稳重感
2. **荧光色**: 禁止使用荧光黄、荧光绿等刺眼颜色
3. **渐变色滥用**: 避免大面积使用渐变色，如需使用应极其克制
4. **多色混搭**: 单页颜色（不含黑白灰）不超过4种

### ⚠️ 谨慎使用

1. **纯黑 (#000000)**: 正文避免使用纯黑，使用 #4A4A4A 更柔和
2. **纯白 (#FFFFFF)**: 大面积背景避免纯白，使用 #F5F0E6 更护眼
3. **红色系**: 除错误提示外，避免使用红色作为主色

## 八、使用建议

1. **优先使用主色**: 80%的情况下使用矿业金(#B89A4B)和深棕褐(#6B4423)进行强调
2. **保持克制**: 单页颜色控制在4种以内（不含黑白灰）
3. **注意对比度**: 文字与背景对比度至少达到4.5:1，确保可读性
4. **统一风格**: 同一汇报材料中保持配色一致，不要混用其他风格
5. **打印适配**: 如需打印，建议使用纯白背景(#FFFFFF)替代浅米白，节省墨水
